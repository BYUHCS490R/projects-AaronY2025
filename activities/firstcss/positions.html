Box Model

1. A block tag is a tag that takes up a whole line. A inline tag is a tag that will only take up as much space as their content/information.

2. Margin is the space outside of the elements border that separates it from the other elements. Padding is the space inside of the elements. The border is between the padding and margin.

3. CSS uses pixels, percent, and ems.

4. You would use a figure tag when I want to use and image.
