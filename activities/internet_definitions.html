Internet

1. The internet is the entire network that connects all of the computers and devices all around the world. It allows people to connect and share information with each other in many different ways including emails, videos, online games, etc.

Web

2. The web is the part of the internet that makes up all of the websites and web pages that we all use. An example of a website could be something like google.com or youtube.com. These are all ways that we can use the Web.

How are the Internet and Web different?

3. The internet and web are different because the Internet is the entire network that houses the Web. The Web is the part of the internet that allows peopele to use websites and web pages. The Internet is what we use to send information to other people. And using a website is part of the Web.

ASCII and Unicode

4. ASCII and Unicode: ASCII is the code that is used to represent the English letters, numbers, etc. Although it is limited because it only supports up to 128 characters. Unicode is the sysyem that represents characters from almost every language in the world, this also includes the emojis and symbols on our computer. Unicode is much bigger and more flexible than ASCII, meaning it probably has a bigger limit to characters.

What is UTF-8?

5. UTF-8 is the storage for Unicode characters. It is the most common text format on the Web. It works for both ASCII and Unicode characters.

URL

6. URL is the address for something on the Web. An example of this would be https://www.youtube.com. It is a url that points to Youtube's homepage.

Domain name

7. Domain name is the name of a website, something that people use to make the websites easier to remember. An example of this is youtube.com.

HTTP

8. HTTP is the system that browsers and servers will use to send and receive web pages for viewing and browsing.

HTTPS

9. HTTPS is the more secure version of HTTP. It uses encryption so that passwords, payments, etc. Are protected while using the internet.

IP Address

10. IP Address is a unique number that identifies each and every single device on the internet. It is basically the phone number for your computers and other devices.

DNS

11. DNS is the internets index. It translates the domain names into the actual IP address for that domain so that you dont have to remember the long string of numbers that show up when you look up a website. If we didnt have DNS we would have to type every single character for the IP address every single time we wanted to visit a website.

W3C Recommendations

12. W3C Recommendations is the official guidelines for how the Web should be working. It stans for World Wide Web Consortium. THe rules make sure that all websites look and work the same across different browsers and different devices.

XML, XHTML, HTML, HTML5

13.
