Structural Tags

  1. The things that go in the header are the title and sometimes navigation links.
  2. Links will go into Nav. The nav has the ability to be anywhere above, inside, or below the header tag. So it does have the ability to go in or outside of the header. The nav does not have a link to a different website, but instead has a link to a different page on the same website.
  3. Any of the main information and content that you may need for the website you are creating.
  4. Any lesser known information that people might want to find. This can include the social media for your website/company. Information about where to find you companies email address, etc.
  5. HTML validation will help you to make sure that you don't have any errors in your code for HTML. You can find it by looking up html validator in your search engine.
